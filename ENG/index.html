Colormash.ru provides ways to create color palettes.

The tool will help you choose a color palette for your site or product. A color palette is a scheme of shades to a particular color. All you have to do is mix the colors to get your palette.

Start picking up the colors in the bottom area of the tool using the "add color" button and the window with the hue setting. Like the color and/or palette you got and don't want to lose it? Just download it from the right micromenu. In the same menu there is a function to display text in different shades to visualize the beautiful design of your site or article. Nearby you can find a button to search for the selected color on Behance.

Explore new colors online with our blending system. Pick up shades or simply type in RGB or HEX code to quickly copy a color you've prepared to create a palette.

If you are completely unfamiliar with color and coloristics, you can check the RAL CLASSIC color table on our site and choose your first shade.
